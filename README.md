### Installing

Download the project by cloning using the command below:

```
git clone https://github.com/codegrann/react-todo-app.git
cd react-to-do-app
```

Install project dependencies and start the app:

```
npm install
npm start
```

or

yarn

```
yarn install
yarn start
```
